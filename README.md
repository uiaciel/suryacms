> **SuryaCMS** adalah Custom CMS berbasis **Laravel** yang modular, ringan, dan scalable, dirancang untuk **developer, agency, dan entrepreneur** yang ingin membangun website profesional tanpa bergantung pada WordPress.

SuryaCMS cocok untuk **Company Profile, Landing Page, Blog, UMKM, Sekolah, hingga Multi Demo Website** dalam satu sistem.

---

## ✨ Fitur Utama

- Laravel 10+ Based CMS
- Modular Package System
- Livewire 3 & Alpine.js
- Multi Theme Support
- Section Builder (Block-based)
- Menu Builder (Nested & Drag Drop)
- Blog & Page Management
- Gallery & Media Manager
- YouTube Video Manager
- Contact Form & Message Inbox
- PWA Ready & Frontend Cache
- SEO Friendly

---

## 📦 Instalasi

### 1. Install via Composer
```bash
composer require uiaciel/suryacms
```

### 2. Publish Configuration & Assets
```bash
php artisan vendor:publish --tag=suryacms-frontend-config
php artisan vendor:publish --tag=suryacms-frontend-views
php artisan vendor:publish --tag=suryacms-frontend-assets
```

### 3. Jalankan Migration
```bash
php artisan migrate
```

### 4. Akses Admin Panel
```text
/login
```
